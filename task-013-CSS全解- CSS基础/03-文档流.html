        理解几个重要的概念
            文档流
                normal flow
                文档流动方向
                从左到右排列 内联元素 例如span 
                从上到下排列 每个元素占一行 例如 div
                在html5规定中不区分块级元素和内联元素 因为可以通过样式来设置
                元素的默认显示只是浏览器的默认显示方式而已 你自己可以修改显示方式

                流动方向
                    inline从左到右
                    block从上到下
                    inline-block 不会像inline元素一样 把元素分成两块显示
                
                宽度
                    inline宽度是内部inline元素的和 不能用width制定
                    block 默认自动计算宽度 可以用width指定
                        div宽度默认auto 不是100% 永远不要写宽度100%
                    inline-block结合前两者特点 可以用width指定
                
                高度
                    inline高度由line-height间接确定 跟height无关
                    block高度由内部文档流元素决定 可以设置height
                    inline-block和block类似 可以设置height
                
                脱离文档流
                    block高度由内部文档流元素决定 可以设置height
                    这句话的意思是不是说 有些元素可以不在文档流中
                
                哪些元素可以脱离文档流
                    float 
                    position: absolute fixed
                
                怎么让元素不脱离文档流
                    不使用上面的属性就不脱离了
            
                overflow移除
                    当内容大于容器
                    当内容的宽度或高度大于容器的 会溢出
                    可用overflow来设置是否显示滚动条
                    
                    auto 灵活设置
                    scroll 永远显示
                    hidden 直接隐藏溢出
                    visible 直接显示溢出部分
                    overflow 可以分为overflow-x overflow-y 
                    
                    
            块 内联 内联快
