        一直回车 出现小气泡式的图形
        
        打开GitHub 在页面填入公钥
        GitHub settings ssh and gpg keys new ssk 
        打开本地生成的id_rsa.pub 将内容填入ssk框内
        
        测试有没有连接成功
        ssh -T git@github.com 
        ssh key 验证身份
        
        推理
            你有一个GitHub账号 一个本地电脑的git仓库 
            GitHub怎么知道电脑和账号都是你的呢
        答案
            每次都输入一次密码 https协议 不方便
            使用ssh key 
            电脑上放私钥 GitHub账号里留下私钥 
            上传代码是用私钥加密 GitHub用公钥解密
            如果解开了 说明是配对的
            同理 你怎么知道对方是GitHub呢 也需要GitHub提供给你一个公钥 
            所以第一次连接GitHub时要输入yes接受对方的公钥
