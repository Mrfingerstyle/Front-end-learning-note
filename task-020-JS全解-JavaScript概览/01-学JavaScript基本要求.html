                    double check 
                抽象思维
                    举例
                        订了一份报纸
                        关注某人微博
                        使用js监听一个按钮的点击事件 
                        这三件事的共同模式 发布订阅模式
                    好处
                        抽象思维使你的代码能解决更多问题
                        而不是一个一个的解决
                        比如Vue React的作者
            硬要求
                足够的代码量
                    统计自己代码量
                        yarn global add cloc
                        在git仓库目录下 执行 cloc --vcs=git . 
                        可以统计一个仓库里的有效代码
                        注意
                            如果仓库里有node_modules目录等不相关内容
                            需要把/node_modules/写入.gitignore文件
