        开始用flex

        container 容器
        items 
            让一个元素变为容器
            .container {
                display: flex;
                display: inline-flex;
            }

        改变items流动方向 主轴
            flex-direction row row-reverse column column-reverse 
        
        flex-wrap 设置是否折行
            flex-wrap 默认不折行
                也可以设置为wrap折行
                一般折行
            wrap-reverse
                从下向上折行 使用较少
        
        主轴对齐方式
            默认 justify-content flex-start 
            flex-end
            center
            
            space-between 
            
            space-around
            
            justify-content: space-evenly;

        次轴对齐方式 当item高度不一样的时候 做上下
            align-items

                stretch

        多行内容
            默认平均分配行高

        实践
            不同布局
                用flex做两栏布局
                用flex做三栏布局
                用flex做四栏布局
                用flex做平均布局
                用flex做综合布局 做更复杂的布局
            
            经验
                永远不要把width height写死 除非特殊说明
                用min-width max-width min-height max-height 
                flex可以基本满足所有需求
                flex和margin-xxx auto配合有意外的效果
