        布局是什么
            把页面分为一块一块 按照左中右 上中下排列
        
        布局分类
            固定宽度布局 一般宽度为960 1000 1024px
            不固定宽度布局 主要靠文档流的原理来布局
        
        文档流本来就是自适应的 不需要加额外的样式
        
        第三种布局
            响应式布局
            意思是PC上固定宽度 手机上不固定宽度
            也就是一种混合布局

        布局的两种思路
            从小到大
                先定下大局
                再完善小布局
            从大到小
                先完成小布局
                然后组合成大布局
            两种均可
                新人推荐第二种
                老手推荐第一种
        
        布局需要什么属性
            div + css 
            css布局 现在使用
