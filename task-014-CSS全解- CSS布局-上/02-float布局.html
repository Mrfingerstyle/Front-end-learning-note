        经验
            有经验者会留一些空间或者最后一个不设width
            不需要做响应式 因为手机上没有IE 而这个布局是专门为IE准备的
            IE6 7 存在双倍margin bug 解决方法
                1 将错就错 针对IE6 7 margin减半
                2 神来一笔 再加一个display inline block 
        
        float不同布局
            float做两栏布局 如顶部条
            float做三栏布局 如内容区
            float做四栏布局 如导航
            float做五栏布局 如产品展示区
            曾经淘宝的前端发明了双飞翼布局 已过时 不要学
        
        经验
            加上头尾  即可满足所有pc页面需求
            手机页面傻子采用float
            float要程序员自己计算宽度 不灵活
            float用来应付IE足矣
        
            有时候border会干扰宽度 因为它会占两像素
            如果有个图层过宽 可以使用margin 负值将块元素向右移动
            float元素的外边距不会合并
