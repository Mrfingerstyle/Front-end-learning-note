                })

                也可以使用forEach map等原型上的函数

                forEach是一个槛
                自己写一个forEach
                原理 遍历数组 每一次都调用那个函数

                function forEach(arr, fn) {
                    for (let i = 0; i < arr.length; i++) {
                        fn(arr[i], i, arr);
                    }
                }
        
                forEach([1, 2, 3], function (x, y, z) {
                    console.log(y, x, z);
                })

                for循环和forEach基本功能一致 
                不同在于break关键字的使用
                前者是块作用域 后者是函数作用域
