        数组变换
            map 
                map([x, x, x], cook)
                对每一项调用函数
                n 变 n 
                一一映射
                不改变原数组 

            filter 
                根据条件过滤元素
                n 变少
                不改变原数组 

            reduce 
                不改变原数组
