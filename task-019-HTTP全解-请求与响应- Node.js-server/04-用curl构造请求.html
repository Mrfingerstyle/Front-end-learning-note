        用curl构造请求
            curl -v http://localhost:8888
            
            设置请求动词
                -X POST 
                注意大小写
                curl -v -X POST http://localhost:8888
            
            设置路径与查询参数
                curl -v -X POST http://localhost:8888/xxxxxx?wd=hi#nihao
                锚点不会发送到服务器
            
            设置请求头
                curl -v -X POST -H 'frank: good' http://localhost:8888/xxxxxx?wd=hi#nihao

            设置请求体
                curl -v -X POST -H 'frank: good' -H 'Content-Type: text/plain;charset=utf-8' -d '请求体' http://localhost:8888/xxxxxx#nihao

        Node.js 读取请求
            读取请求动词
                request.method 
            读取路径
                request.url 路径 带查询参数
                path 路径 不带查询参数
                query 只有查询参数
            读取请求头
                request.headers['Accept']
            读取请求体
                比较复杂 后续详解
            设置状态码
                response.statusCode=200
            设置响应头
                response.setHeader('Content-Type', 'text/html')
            设置响应头
                response.write('内容')
                可追加内容
                response.write() 内容可以追加
            目的是搞清楚响应 请求
