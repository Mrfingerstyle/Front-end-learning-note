        推荐 Node 8 10 12
        Node安装需要将其加入环境变量
        安装之后有了以下命令
            node
            npm
            npx
        配置node
            安装nrm
                npm i -g nrm 
                显示可用镜像 nrm ls
                nrm use taobao 选择镜像源
                此处使用淘宝源
            安装yarn 代替npm bug少 速度快
                官网下载安装包 添加到系统环境变量
                查看yarn配置
                    yarn config list
                yarn安装yrm
                    yarn global add yrm 
                查看可用镜像
                    yrm ls
                设置镜像
                    yrm use taobao 
                nrm的淘宝源改动会自动作用到yarn上 使用yarn config list 查看registry
                软件安装之后都需要配置一下
